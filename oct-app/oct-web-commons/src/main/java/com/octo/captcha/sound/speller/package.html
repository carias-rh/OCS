This package a struture for implementing spellerSound
A gimpySound is SoundCaptcha with the question "Write the word spelled out". GimpySound is based on the human ability to listen word spelled and write it down, and the diffuculty of current computer programs to do the same. This package defines a new com.octo.captcha.CaptchaFactory :
  • SpellerSoundFactory : a com.octo.captcha.CaptchaFactory building SpellerSounds from words
